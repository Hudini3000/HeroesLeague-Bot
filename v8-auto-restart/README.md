# 勇者联盟首领挂机脚本 v8

## 🎯 版本特点

**v8 核心升级：凌晨 4 点自动重启**

- ✅ 每天凌晨 4:00-4:05 自动退出脚本
- ✅ 退出前自动回到主界面（更像人类操作）
- ✅ 守护进程（watchdog.bat）自动重启脚本
- ✅ 24/7 无人值守挂机
- ✅ 释放内存，避免 Python 长时间运行内存泄漏

**继承自 v7 的功能：**

- HSV 橙色检测（"已刷新"标签）
- 多尺度模板匹配（适配不同窗口大小）
- 战斗结束检测（等"确定"按钮出现）
- F2=暂停/继续  F4=停止

---

## 🚀 快速开始

### 方法一：守护进程模式（推荐，24/7 挂机）

1. **双击 `watchdog.bat`**
2. 脚本会启动，守护进程在后台运行
3. 每天凌晨 4 点，脚本自动退出并重启
4. 最小化 watchdog 窗口即可

**停止守护进程：** 在 watchdog 窗口按 `Ctrl + C`

---

### 方法二：手动模式（不自动重启）

1. **双击 `start_v8.bat`**
2. 脚本启动，退出后不自动重启
3. 适合需要手动控制重启的场景

---

## ⚙️ 配置说明

### 自动重启时间

在 `boss_auto_v8.py` 中修改：

```python
CONFIG = {
    "restart_hour": 4,             # 每天几点自动重启（24小时制）
    "restart_window": 5,           # 重启时间窗口（分钟），比如4:00-4:05
}
```

**建议：**
- 凌晨 4 点：大多数玩家不在线，服务器负载低
- 如果想改时间，比如凌晨 3 点，改成 `"restart_hour": 3`

---

### 守护进程日志

watchdog.bat 会记录重启日志到 `watchdog_log.txt`：

```
[2026-06-28 04:00:01.23] 守护进程启动
[2026-06-28 04:00:02.45] 正在启动脚本...
[2026-06-28 04:05:12.67] 脚本已退出，等待10秒后重启...
```

## 📋 使用流程

### 第一次使用

1. 打开游戏，进入首领列表界面
2. 双击 `watchdog.bat`
3. 脚本开始运行，看到日志输出
4. 按 `F2` 可暂停，按 `F4` 可停止

### 每天凌晨 4 点

1. 脚本检测到时间是 4:00-4:05
2. 自动按 `ESC` 回到主界面
3. 脚本退出（退出码 0）
4. watchdog.bat 检测到脚本退出
5. 等待 10 秒
6. 自动重启脚本
7. 脚本重新进入首领列表，继续挂机

## 🔧 故障排查

### 守护进程不重启

**原因：** 脚本不是正常退出（比如崩溃）

**解决：**
1. 检查 `watchdog_log.txt` 看错误信息
2. 手动双击 `watchdog.bat` 重启守护进程
3. 检查脚本是否有语法错误（运行 `start_v8.bat` 看报错）

---

### 脚本在 4 点没有重启

**原因1：** 脚本在暂停状态（按了 F2）

**解决：** 确保脚本在运行状态（没按 F2）

**原因2：** 今天已经重启过了

**解决：** 删除 `.restart_marker` 文件，强制重启

```powershell
Remove-Item "C:\Users\Administrator\Desktop\HeroesLeague-Bot\v8-auto-restart\.restart_marker"
```

---

### 游戏掉线后脚本卡住

**原因：** 脚本一直在等"确定"按钮

**解决：**
1. 按 `F4` 停止脚本
2. 重新登录游戏
3. 双击 `watchdog.bat` 重启

**建议：** 可以加一个"检测游戏是否掉线"的功能（未来版本）
