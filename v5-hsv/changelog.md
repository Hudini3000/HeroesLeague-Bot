# v5 HSV颜色检测 · 自动滚屏版

**日期：** 2026-06-27
**作者：** QClaw

---

## 核心改进

### 🟢 HSV 颜色检测代替文字模板匹配
- 不再依赖 `refreshed_text.png` 模板
- 直接扫描绿色像素（H:40-75 S:120+ V:130+）找到"已刷新"标签
- 用**百分比过滤**，自动适配任意窗口大小/分辨率

### 📜 自动滚屏
- 检测到列表底部没有更多 BOSS 后，尝试向上滚屏继续扫描
- 最多支持识别上下滚动的主界首领列表

### 🐛 修复问题
- 窗口大小变化导致文字模板失效 → HSV 不受影响
- 720x1280 / 360x640 / 1440x2560 三种分辨率测试 100% 准确

## 文件说明

| 文件 | 用途 |
|------|------|
| `boss_auto_v5.py` | 主脚本（当前版本） |
| `boss_config.json` | 颜色和区域配置 |
| `templates/` | 模板图片（首领列表按钮/确认按钮/前往按钮） |
| `capture_boss_templates.py` | 重新截取模板 |
| `boss_auto_v4_reference.py` | v4 参考（HSV 前的版本） |
| `boss_auto_v1_reference.py` | v1 参考（最早版本） |

---

## 使用方法

1. 双击 `首领挂机v5.bat`
2. 启动后按 **F1** 初始化（确认窗口位置）
3. 按 **F2** 暂停/继续
4. 按 **F4** 停止

---

## 热键

| 按键 | 功能 |
|------|------|
| F1 | 初始化（重新定位游戏窗口） |
| F2 | 暂停 / 恢复 |
| F4 | 强制停止 |
